- LunaBeat ต้องการมอบประสบการณ์เพลงท้องถิ่นที่สะอาดและดื่มด่ำยิ่งขึ้น เหมือนเสียงเพลงยามค่ำคืน ระหว่างทำนองและเนื้อเพลง
---
- LunaBeat คือเครื่องเล่นเพลงบนมือถือที่เน้นเพลงท้องถิ่นและประสบการณ์เนื้อเพลง รวมการเล่นเพลง การแก้ไขเนื้อเพลง และการจับเวลาแบบรายคำไว้ด้วยกัน
- รองรับการเล่นไฟล์เสียงท้องถิ่น การสร้างเนื้อเพลงรายคำรวมถึงดูเอตและเสียงพื้นหลัง การพรีวิวและแก้ไขเนื้อเพลง ส่งออกเป็น `.lrc` / `.elrc` / `.ttml` และรองรับการประมวลผลเมตาดาต้าเสียงกับเนื้อเพลงแบบกลุ่ม
- ตั้งแต่นำเข้า แก้ไข เล่น ไปจนถึงส่งออก LunaBeat มุ่งสร้างประสบการณ์เพลงและเนื้อเพลงท้องถิ่นที่ครบถ้วนและดื่มด่ำกว่าเดิม

## ทำไมต้องใช้ LunaBeat

- ไม่ใช่แค่ตัวแก้ไขเนื้อเพลง แต่ยังใช้เป็นเครื่องเล่นเพลงท้องถิ่นในระยะยาวได้
- ทำการจับเวลาเนื้อเพลงรายคำระดับมืออาชีพได้บนมือถือ
- รองรับการนำเข้าและส่งออกหลายรูปแบบ เหมาะกับเวิร์กโฟลว์เนื้อเพลงทั่วไป
- รวมคลังเพลงและเครื่องมือเมตาดาต้า ลดการสลับไปมาระหว่างแอป
- มีการประมวลผลแบบกลุ่ม เหมาะสำหรับจัดระเบียบเพลงจำนวนมาก

## ภาพรวมฟีเจอร์

### 1. การจับเวลาและแก้ไขเนื้อเพลง
- การจับเวลาด้วย 3 ปุ่ม: เริ่ม / ต่อเนื่อง / จบ
- ย้อนกลับ / กรอไปข้างหน้า, เล่น / หยุดชั่วคราว, ความเร็ว 0.25x ~ 3x และข้ามไปยังเวลาที่ระบุ
- เลิกทำ / ทำซ้ำ สำหรับเวลา เนื้อเพลง และการจัดการบรรทัด
- แตะเวลาเล่นเพื่อเข้าสู่โหมดติดตามและไปยังเนื้อเพลงที่เลือก
- แตะหน่วยเนื้อเพลงเพื่อเลือก แตะสองครั้งเพื่อข้ามไปยังเวลาเริ่มของเนื้อเพลงปัจจุบันและเล่น

### 2. นำเข้า
- เนื้อเพลงข้อความธรรมดา
- LRC (รายบรรทัด / รายคำ)
- Enhanced LRC (ELRC)
- TTML
- ค้นหาเนื้อเพลงจากบริการสตรีมมิง
- นำเข้าไฟล์เสียง

### 3. ส่งออก
- LRC (รายบรรทัด / รายคำ)
- TTML (รองรับดูเอตและเสียงพื้นหลัง)

### 4. พรีวิวและประมวลผลแบบกลุ่ม
- พรีวิวเอฟเฟกต์การเล่นเนื้อเพลงรายคำ
- แยกคำในคลิกเดียว
- เลื่อน timestamp
- รวมเนื้อเพลงในคลิกเดียว
- แปลงจีนตัวเต็มเป็นจีนตัวย่อ
- ลบบรรทัดว่างในคลิกเดียว

### 5. การจัดการโครงสร้างเนื้อเพลง (กดค้าง)
- จัดการเนื้อเพลง: แก้ไข เพิ่ม แยก รวม ตั้งเวลา ลบ
- จัดการบรรทัด: เพิ่ม รวม แยก ย้าย เพิ่ม/แก้คำแปล ลบ

### 6. เครื่องเล่น
- เล่นเพลงท้องถิ่น รองรับเล่น / หยุดชั่วคราว เพลงก่อนหน้า / ถัดไป และลากตำแหน่งเล่น
- เล่นตามลำดับ สุ่มเล่น วนซ้ำเพลงเดียว
- จัดการคิวที่กำลังเล่น รองรับข้าม ลบ และลากเพื่อเรียงลำดับ
- เชื่อมต่อมินิเพลเยอร์ในคลังเพลงกับเครื่องเล่นเต็มรูปแบบ
- เล่นเบื้องหลัง การแจ้งเตือนสื่อของระบบ และควบคุมด้วยปุ่มมีเดีย
- ตั้งเวลาปิด รองรับหยุดตามเวลานับถอยหลังหรือหยุดเมื่อเพลงปัจจุบันจบ
- เชื่อมกับพรีวิวเนื้อเพลงเพื่อดูเอฟเฟกต์รายคำขณะเล่น
- รองรับรูปแบบเสียงท้องถิ่นทั่วไป และมีการจัดการความเข้ากันได้สำหรับ ALAC

### 7. คลังเพลง
- สแกนและกรองไฟล์เสียงท้องถิ่น
- เพลย์ลิสต์ เพลงโปรด และเพลงห้าดาว
- นำเข้า / ส่งออกเพลย์ลิสต์
- เพิ่มเข้าคิว / เล่นถัดไป
- จับคู่แท็กแบบกลุ่ม เช่น ปก ชื่อเพลง ศิลปิน และอื่น ๆ
- จับคู่เนื้อเพลงแบบกลุ่ม ทั้งรายคำ / รายบรรทัด / คำแปล
- เปลี่ยนชื่อแบบกลุ่ม
- แก้ไขแท็กแบบกลุ่ม

### 8. การตั้งค่า
- โหมดมืดแบบกำหนดเอง
- กำหนดเวลาย้อนกลับ / กรอไปข้างหน้าสำหรับการจับเวลาเนื้อเพลง
- การตั้งค่าคลังเพลง
- รูปแบบและพื้นหลังแถบควบคุมการเล่น
- การแสดงเนื้อเพลง เนื้อเพลงบนเดสก์ท็อป เนื้อเพลงแถบสถานะ และเนื้อเพลงผ่าน Bluetooth ในรถ
- ตั้งค่าขนาดปกของเมตาดาต้าเสียง
- ตั้งค่าภูมิภาคเมตาดาต้า Apple Music

## เริ่มใช้งานใน 5 นาที (ลำดับที่แนะนำ)

### ขั้นตอนที่ 1: นำเข้าเสียงและเนื้อเพลง
1. นำเข้าไฟล์เสียงก่อน
2. จากนั้นเลือกวิธีนำเข้าตามแหล่งเนื้อเพลง (ข้อความธรรมดา / LRC / ELRC / TTML / ค้นหา)

### ขั้นตอนที่ 2: เล่นเพลงท้องถิ่น
1. เข้าคลังเพลงและสแกนไฟล์เสียงท้องถิ่น
2. แตะเพลงเพื่อเริ่มเล่น มินิเพลเยอร์ด้านล่างจะแสดงสถานะการเล่นปัจจุบัน
3. เข้าเครื่องเล่นเต็มรูปแบบเพื่อสลับโหมดเล่น จัดการคิว เปิดพรีวิวเนื้อเพลง หรือตั้งเวลาปิด

### ขั้นตอนที่ 3: เตรียมเนื้อเพลง
1. ใช้การแยกคำในคลิกเดียวเพื่อให้ได้หน่วยรายคำ
2. ใช้การลบบรรทัดว่าง รวมเนื้อเพลง หรือแปลงจีนตัวเต็มเป็นตัวย่อตามความจำเป็น
3. หากมีไทม์ไลน์อยู่แล้วแต่เลื่อนทั้งชุด ให้เลื่อน timestamp ก่อนแล้วค่อยปรับละเอียด

### ขั้นตอนที่ 4: เริ่มจับเวลา (3 ปุ่มหลัก)
- เริ่ม: ตั้งเวลาเริ่มของหน่วยปัจจุบัน ไม่ข้ามไป
- ต่อเนื่อง: ตั้งเวลาจบของหน่วยปัจจุบันและไปยังหน่วยถัดไปอัตโนมัติ พร้อมเขียนเวลาเริ่มของหน่วยถัดไป
- จบ: ตั้งเวลาจบของหน่วยปัจจุบันและข้ามไปหน่วยถัดไป แต่ไม่เขียนเวลาเริ่มของหน่วยถัดไป

จังหวะที่แนะนำ: `เริ่ม -> ต่อเนื่อง -> ต่อเนื่อง -> ... -> จบ` วนตามแต่ละประโยค

### ขั้นตอนที่ 5: ปรับเวลาและข้อความละเอียด
1. เปิดโหมดติดตาม เพื่อให้การเล่นระบุตำแหน่งหน่วยปัจจุบันอัตโนมัติ
2. แตะเนื้อเพลงสองครั้งเพื่อข้ามกลับไปยังเวลาเริ่มและฟังทวน
3. ใช้เลิกทำ / ทำซ้ำเพื่อเปรียบเทียบผลก่อนและหลังแก้ไขอย่างรวดเร็ว
4. กดค้างที่เนื้อเพลงหรือบรรทัดเพื่อแยก รวม ย้าย แปล หรือแก้โครงสร้าง

### ขั้นตอนที่ 6: ตั้งค่าดูเอตและคำแปล (ไม่บังคับ)
- ประเภทดูเอต: ดูเอตซ้าย (เสียงหลัก), ดูเอตขวา (เสียงรอง), พื้นหลัง (เสียงพื้นหลัง)
- คำแปล: รองรับการนำเข้าหรือแก้ไขด้วยตนเอง
- หมายเหตุ: ประเภทดูเอตมีผลเฉพาะในเวิร์กโฟลว์ TTML

### ขั้นตอนที่ 7: พรีวิวและส่งออก
1. ตรวจเอฟเฟกต์การเล่นรายคำในหน้าพรีวิวเนื้อเพลง
2. ส่งออกตามการใช้งาน:
   - เครื่องเล่นทั่วไปแนะนำ `LRC`
   - หากต้องการข้อมูลดูเอต / เสียงพื้นหลัง แนะนำ `TTML`

## คำอธิบายหน้าจอจับเวลา

- คอลัมน์: เวลาเริ่ม / เนื้อเพลง / เวลาจบ / คำแปล
- ระบุตำแหน่ง: ไปยังเนื้อเพลงที่เลือกได้
- ข้ามเวลา: แตะเวลาในพื้นที่จับเวลาเพื่อข้ามไปยังเวลาที่กำหนดเอง
- แตะชื่อ: รับพาธไฟล์เสียงได้

## เวิร์กโฟลว์คลังเพลงที่แนะนำ

1. สแกนไฟล์เสียงท้องถิ่น ตรวจสอบและกรองเพลงเป้าหมาย
2. จัดเพลงที่ฟังบ่อยตามอัลบั้ม เพลย์ลิสต์ รายการโปรด หรือเพลงห้าดาว
3. เล่นเพลงและจัดการคิวปัจจุบัน หากจำเป็นให้ตั้งเล่นถัดไปหรือตั้งเวลาปิด
4. จับคู่แท็กแบบกลุ่มเพื่อเติมปก / ชื่อเพลง / ศิลปิน และเมตาดาต้าอื่น ๆ
5. จับคู่เนื้อเพลงแบบกลุ่มเพื่อเติมรายคำ / รายบรรทัด และคำแปล
6. เปลี่ยนชื่อและแก้ไขแท็กแบบกลุ่มเพื่อจัดระเบียบให้เสร็จ

## FAQ

### 1. ควรส่งออก LRC เมื่อไร และ TTML เมื่อไร?
- หากต้องการแสดงเนื้อเพลงทั่วไป เลือก LRC
- หากต้องการดูเอต เสียงพื้นหลัง หรือโครงสร้างที่ครบกว่า เลือก TTML

### 2. ทำไมตั้งค่าดูเอตแล้วไม่เห็นเอฟเฟกต์?
ประเภทดูเอตรองรับเฉพาะ TTML โปรดยืนยันว่าใช้งานเวิร์กโฟลว์ TTML และส่งออกเป็น TTML

### 3. เนื้อเพลงเร็วหรือช้าทั้งชุด จะแก้เร็วอย่างไร?
ใช้การเลื่อน timestamp เพื่อปรับทั้งชุดก่อน แล้วกลับไปที่หน้าจับเวลาเพื่อปรับละเอียดเฉพาะจุด

### 4. มือใหม่มักพลาดขั้นตอนไหน?
มักเป็นการจับเวลาก่อนแยกคำ แนะนำให้แยกคำและจัดข้อความให้เรียบร้อยก่อนเริ่มจับเวลา
