- LunaBeat는 밤의 음악처럼 멜로디와 가사 사이에서 더 맑고 몰입감 있는 로컬 음악 경험을 제공하고자 합니다.
---
- LunaBeat는 로컬 음악과 가사 경험에 집중한 모바일 플레이어로, 음악 재생, 가사 편집, 글자 단위 타이밍 작업을 하나로 결합합니다.
- 로컬 오디오 재생, 듀엣과 배경 보컬을 포함한 글자 단위 가사 제작, 가사 미리보기와 편집을 지원하며 `.lrc` / `.elrc` / `.ttml`로 내보낼 수 있고, 오디오 메타데이터와 가사 일괄 처리도 지원합니다.
- 가져오기, 편집, 재생, 내보내기까지 LunaBeat는 더 완성도 높고 몰입감 있는 로컬 음악과 가사 경험을 제공합니다.

## LunaBeat를 사용하는 이유

- 단순한 가사 편집기를 넘어 오래 사용할 수 있는 로컬 음악 플레이어입니다.
- 모바일에서 전문적인 글자 단위 가사 타이밍 작업을 할 수 있습니다.
- 여러 형식의 가져오기와 내보내기를 지원하여 일반적인 가사 작업 흐름에 맞습니다.
- 음악 라이브러리와 메타데이터 도구를 통합해 앱을 오가는 번거로움을 줄입니다.
- 많은 곡을 정리하기 좋은 일괄 처리 기능을 제공합니다.

## 기능 개요

### 1. 가사 타이밍 및 편집
- 시작 / 연속 / 종료 3키 타이밍 작업
- 되감기 / 빨리감기, 재생 / 일시정지, 0.25x ~ 3x 배속, 지정 시간으로 이동
- 시간, 가사, 줄 작업에 대한 실행 취소 / 다시 실행
- 재생 시간을 눌러 따라가기 모드로 전환하고 선택한 가사로 이동
- 가사 단위를 탭해 선택하고, 두 번 탭해 현재 가사 시작 시간으로 이동해 재생

### 2. 가져오기
- 일반 텍스트 가사
- LRC(줄 단위 / 글자 단위)
- 향상된 LRC(ELRC)
- TTML
- 스트리밍 가사 검색
- 오디오 파일 가져오기

### 3. 내보내기
- LRC(줄 단위 / 글자 단위)
- TTML(듀엣, 배경 보컬 지원)

### 4. 미리보기 및 일괄 처리
- 글자 단위 재생 효과 미리보기
- 원클릭 단어 분리
- 타임스탬프 이동
- 원클릭 가사 병합
- 번체 중국어를 간체 중국어로 변환
- 원클릭 빈 줄 삭제

### 5. 가사 구조 작업(길게 누르기)
- 가사 작업: 편집, 추가, 분할, 병합, 시간 설정, 삭제
- 줄 작업: 추가, 병합, 분할, 이동, 번역 추가/수정, 삭제

### 6. 플레이어
- 로컬 음악 재생, 재생 / 일시정지, 이전 곡 / 다음 곡, 진행 위치 이동
- 순차 재생, 셔플 재생, 한 곡 반복
- 현재 재생 대기열 관리, 이동 재생, 제거, 드래그 정렬
- 음악 라이브러리 미니 플레이어와 전체 플레이어 연동
- 백그라운드 재생, 시스템 미디어 알림, 미디어 키 제어
- 카운트다운 종료 또는 현재 곡 종료 후 정지를 지원하는 수면 타이머
- 가사 미리보기와 연동하여 재생 중 글자 단위 가사 효과 확인
- 일반적인 로컬 오디오 형식을 지원하며 ALAC 재생 호환 처리를 제공합니다

### 7. 음악 라이브러리
- 로컬 오디오 스캔 및 필터링
- 내 재생목록, 좋아요 곡, 별 다섯 개 곡
- 재생목록 가져오기 / 내보내기
- 재생 대기열에 추가 / 다음 곡으로 재생
- 커버, 제목, 아티스트 등 태그 일괄 매칭
- 글자 단위 / 줄 단위 / 번역 가사 일괄 매칭
- 파일 일괄 이름 변경
- 태그 일괄 편집

### 8. 설정
- 사용자 지정 다크 모드
- 가사 타이밍용 되감기 / 빨리감기 시간 설정
- 음악 라이브러리 설정
- 재생 바 스타일과 배경
- 가사 표시, 데스크톱 가사, 상태 표시줄 가사, 차량 Bluetooth 가사
- 오디오 메타데이터 커버 크기 설정
- Apple Music 메타데이터 지역 설정

## 5분 빠른 시작(권장 순서)

### 1단계: 오디오와 가사 가져오기
1. 먼저 오디오 파일을 가져옵니다.
2. 그런 다음 가사 출처에 따라 가져오기 방식(일반 텍스트 / LRC / ELRC / TTML / 검색)을 선택합니다.

### 2단계: 로컬 음악 재생
1. 음악 라이브러리에 들어가 로컬 오디오를 스캔합니다.
2. 곡을 탭해 재생을 시작하면 하단 미니 플레이어가 현재 재생 상태를 표시합니다.
3. 전체 플레이어에서 재생 모드 변경, 대기열 관리, 가사 미리보기 열기, 수면 타이머 설정을 할 수 있습니다.

### 3단계: 가사 전처리
1. 원클릭 단어 분리로 글자 단위 유닛을 만듭니다.
2. 필요하면 빈 줄 삭제, 가사 병합, 번체-간체 변환을 사용합니다.
3. 기존 타임라인에 전체 오프셋이 있으면 먼저 타임스탬프 이동 후 세부 조정합니다.

### 4단계: 타이밍 시작(핵심 3키)
- 시작: 현재 유닛의 시작 시간을 설정하며 이동하지 않습니다.
- 연속: 현재 유닛의 종료 시간을 설정하고 다음 유닛으로 자동 이동하며, 다음 유닛의 시작 시간도 기록합니다.
- 종료: 현재 유닛의 종료 시간을 설정하고 다음 유닛으로 이동하지만, 다음 유닛의 시작 시간은 기록하지 않습니다.

권장 리듬: `시작 -> 연속 -> 연속 -> ... -> 종료`를 문장 단위로 반복합니다.

### 5단계: 시간과 텍스트 세부 조정
1. 따라가기 모드를 켜면 재생 중 현재 유닛을 자동으로 찾습니다.
2. 가사를 두 번 탭해 해당 시작 시간으로 빠르게 이동해 다시 들어봅니다.
3. 실행 취소 / 다시 실행으로 수정 전후 효과를 빠르게 비교합니다.
4. 가사나 줄을 길게 눌러 분할, 병합, 이동, 번역 등 구조를 수정합니다.

### 6단계: 듀엣과 번역 설정(선택)
- 듀엣 유형: 왼쪽 듀엣(메인 보컬), 오른쪽 듀엣(보조 보컬), 배경(배경 보컬).
- 번역은 가져오거나 직접 편집할 수 있습니다.
- 참고: 듀엣 유형은 TTML 작업 흐름에서만 적용됩니다.

### 7단계: 미리보기 및 내보내기
1. 가사 미리보기 페이지에서 글자 단위 재생 효과를 확인합니다.
2. 용도에 따라 내보냅니다:
   - 일반 플레이어에는 `LRC` 우선
   - 듀엣 / 배경 보컬 정보가 필요하면 `TTML` 우선

## 타이밍 화면 설명

- 열 정보: 시작 시간 / 가사 / 종료 시간 / 번역
- 위치 이동: 선택한 가사로 이동 가능
- 시간 이동: 타이밍 영역의 시간을 눌러 원하는 시간으로 이동
- 제목 탭: 오디오 경로 확인

## 음악 라이브러리 작업 흐름 제안

1. 로컬 오디오를 스캔하고 대상 곡을 확인 및 필터링합니다.
2. 앨범, 재생목록, 좋아요, 별 다섯 개 곡으로 자주 듣는 음악을 정리합니다.
3. 곡을 재생하고 현재 대기열을 관리하며, 필요하면 다음 곡 재생이나 수면 타이머를 설정합니다.
4. 태그를 일괄 매칭해 커버 / 제목 / 아티스트 등 메타데이터를 보완합니다.
5. 가사를 일괄 매칭해 글자 단위 / 줄 단위 / 번역을 채웁니다.
6. 일괄 이름 변경과 태그 편집으로 최종 정리를 마칩니다.

## FAQ

### 1. 언제 LRC를 내보내고 언제 TTML을 내보내야 하나요?
- 일반 가사 표시만 필요하면 LRC를 선택하세요.
- 듀엣, 배경 보컬, 더 완전한 구조가 필요하면 TTML을 선택하세요.

### 2. 듀엣을 설정했는데 효과가 보이지 않는 이유는 무엇인가요?
듀엣 유형은 TTML에서만 지원됩니다. 현재 TTML 작업 흐름인지 확인하고 TTML로 내보내세요.

### 3. 가사가 전체적으로 빠르거나 늦습니다. 빠르게 수정하려면?
먼저 타임스탬프 이동으로 전체를 맞춘 뒤, 타이밍 페이지로 돌아가 부분적으로 세부 조정하는 것이 효율적입니다.

### 4. 초보자가 가장 자주 실수하는 단계는 무엇인가요?
대개 "먼저 타이밍을 찍고 나중에 단어 분리"하는 것입니다. 먼저 단어 분리와 텍스트 정리를 마친 뒤 타이밍을 시작하는 것을 권장합니다.
