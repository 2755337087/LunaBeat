- LunaBeat 希望像夜晚中的音樂一樣，在旋律與歌詞之間，帶來更純淨、更沉浸的本地音樂體驗。
---
- LunaBeat 是一款專注於本地音樂與歌詞體驗的行動端播放器，融合音樂播放、歌詞編輯與逐字打軸功能於一體。
- 支援本地音訊播放、逐字歌詞製作（含對唱與背景聲）、歌詞預覽與編輯，可匯出 `.lrc` / `.elrc` / `.ttml`，並支援音訊中繼資料與歌詞批次處理。
- 從匯入、編輯、播放到匯出，LunaBeat 致力於提供更完整、更沉浸的本地音樂與歌詞體驗。

## 為什麼用 LunaBeat

- 不只是歌詞編輯器，也可作為本地音樂播放器長期使用。
- 行動端即可完成專業級逐字打軸。
- 支援多格式匯入匯出，適配常見歌詞工作流程。
- 整合音樂庫與中繼資料工具，減少來回切換軟體的成本。
- 內建批次處理能力，適合整理大量歌曲。

## 功能總覽

### 1. 打軸與編輯
- 起始 / 連續 / 結束三鍵打軸
- 快退 / 快進、播放 / 暫停、倍速（0.25x ~ 3x）、跳轉指定時間
- 復原 / 重做（涵蓋時間、歌詞、行操作）
- 點擊播放時間可進入跟隨模式、定位到選中歌詞
- 單擊歌詞單元選中，雙擊跳轉目前歌詞開始時間播放

### 2. 匯入
- 純文字歌詞
- LRC（逐行 / 逐字）
- 增強 LRC（ELRC）
- TTML
- 串流平台歌詞搜尋
- 音訊檔案匯入

### 3. 匯出
- LRC（逐行 / 逐字）
- TTML（支援對唱、和聲）

### 4. 預覽與批次處理
- 逐字播放效果預覽
- 一鍵分詞
- 時間戳平移
- 一鍵合併歌詞
- 繁體轉簡體
- 一鍵刪除空行

### 5. 歌詞結構操作（長按）
- 歌詞操作：編輯、新增、拆分、合併、設定時間、刪除
- 行操作：新增、合併、拆分、移動、加入 / 修改翻譯、刪除

### 6. 播放器
- 本地音樂播放，支援播放 / 暫停、上一首 / 下一首、進度拖曳
- 順序播放、隨機播放、單曲循環
- 目前播放佇列管理，支援跳轉、移除與拖曳排序
- 音樂庫迷你播放器與完整播放器聯動
- 背景播放、系統媒體通知與媒體按鍵控制
- 定時播放，支援倒數停止或播放完目前歌曲後停止
- 與歌詞預覽聯動，播放時可查看逐字歌詞效果
- 支援常見本地音訊格式，並針對 ALAC 做播放相容處理

### 7. 音樂庫
- 掃描本地音訊並篩選
- 我的歌單、收藏歌曲、五星歌曲
- 歌單匯入 / 匯出
- 加入播放佇列 / 下一首播放
- 批次匹配標籤（封面、標題、演出者等）
- 批次匹配歌詞（逐字 / 逐行 / 翻譯）
- 批次重新命名
- 批次編輯標籤

### 8. 設定
- 自訂深色模式
- 自訂打軸快進 / 快退時長
- 音樂庫設定
- 播放控制列樣式與背景
- 歌詞顯示、桌面歌詞、狀態列歌詞與車載藍牙歌詞
- 音訊中繼資料封面尺寸設定
- Apple Music 中繼資料地區設定

## 5 分鐘快速上手（建議順序）

### 第 1 步：匯入音訊與歌詞
1. 先匯入音訊檔案。
2. 再按歌詞來源選擇匯入方式（純文字 / LRC / ELRC / TTML / 搜尋）。

### 第 2 步：播放本地音樂
1. 進入音樂庫並掃描本地音訊。
2. 點擊歌曲開始播放，底部迷你播放器會顯示目前播放狀態。
3. 進入完整播放器，可切換播放模式、管理佇列、開啟歌詞預覽或設定定時播放。

### 第 3 步：預處理歌詞
1. 使用一鍵分詞，得到逐字單元。
2. 視情況使用一鍵刪除空行、合併歌詞、繁轉簡。
3. 如果已有時間軸但整體偏移，先做時間戳平移再進入精修。

### 第 4 步：開始打軸（核心三鍵）
- 起始：設定目前單元開始時間，不跳轉。
- 連續：設定目前單元結束時間並自動到下一單元，同時寫入下一單元開始時間。
- 結束：設定目前單元結束時間並跳到下一單元，但不寫下一單元開始時間。

推薦節奏：`起始 -> 連續 -> 連續 -> ... -> 結束`，按句循環。

### 第 5 步：精修時間與文字
1. 開啟跟隨模式，播放時自動定位目前單元。
2. 雙擊歌詞快速跳到該處開始時間回聽。
3. 用復原 / 重做快速對比修改前後效果。
4. 長按歌詞或行，做拆分 / 合併 / 移動 / 翻譯等結構修正。

### 第 6 步：設定對唱與翻譯（可選）
- 對唱類型：左對唱（主唱）、右對唱（副唱）、背景（和聲）。
- 翻譯：支援匯入或手動編輯。
- 注意：對唱類型僅在 TTML 工作流程中生效。

### 第 7 步：預覽並匯出
1. 在歌詞預覽頁檢查逐字播放效果。
2. 根據用途匯出：
   - 通用播放器優先 `LRC`
   - 需要對唱 / 和聲資訊優先 `TTML`

## 打軸介面說明

- 欄位資訊：開始時間 / 歌詞 / 結束時間 / 翻譯
- 定位：可定位到選中歌詞
- 時間跳轉：點擊打軸區域時間可自訂跳轉
- 標題點擊：可取得音訊路徑

## 音樂庫工作流程建議

1. 掃描本地音訊，先清點與過濾目標歌曲。
2. 按專輯、歌單、收藏或五星歌曲整理常聽內容。
3. 播放歌曲並管理目前佇列，必要時設定下一首播放或定時播放。
4. 批次匹配標籤，補全封面 / 標題 / 演出者等中繼資料。
5. 批次匹配歌詞，統一補齊逐字 / 逐行與翻譯。
6. 批次重新命名與批次編輯，完成最終整理。

## 常見問題（FAQ）

### 1. 什麼時候匯出 LRC，什麼時候匯出 TTML？
- 只需要常規歌詞顯示：選 LRC。
- 需要對唱、和聲、更完整結構：選 TTML。

### 2. 為什麼我設定了對唱但看不到效果？
對唱類型僅支援 TTML。請確認目前是 TTML 工作流程，並以 TTML 匯出。

### 3. 歌詞整體早了 / 晚了，如何快速修正？
先用時間戳平移做整體校準，再回到打軸頁做局部精修，效率最高。

### 4. 新手最容易出錯的步驟是什麼？
通常是「先打軸後分詞」。建議先完成分詞與文字清理，再開始打軸。
